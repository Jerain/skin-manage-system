# AI语音助手 - 需求概述

在客户端和门店端小程序中增加AI语音助手，接入大模型，支持语音或文字交互，可操作门店管理系统所有功能。

## 双模式设计
- 模式一：语音交互模式 - 通过AI语音助手完成所有操作
- 模式二：手动操作模式 - 传统按钮/列表操作方式
- 支持自由切换，记住用户习惯

## 核心能力
**客户端**：预约服务、门店查询、服务项目、订单查询、会员服务
**门店端（技师）**：客户管理、预约管理、考勤打卡、业绩查询、数据看板
**管理后台（店长）**：门店管理、员工管理、数据统计、预约管理

## 技术方案
- 前端：uni-app
- 语音识别：微信同声传译插件 / 阿里云ASR
- 大模型：GLM-4.6
- 对话管理：自研意图识别 + RAG

## 3D形象设计
首页中央展示3D卡通女性AI助手形象，支持：
- 待机状态：自然呼吸动画
- 说话状态：嘴唇同步动效
- 倾听状态：眼神注视动画
- 思考状态：头部轻微晃动

## 开发计划
- Phase 1（2周）：基础能力 - 3D形象、语音识别/合成、对话接口
- Phase 2（2周）：业务对接 - 预约、查询、订单、会员
- Phase 3（2周）：高级能力 - 门店端、后台、RAG优化
